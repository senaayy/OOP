Method Overriding (Metot Ezme)

Method overriding, bir alt sınıfın üst sınıftan miras aldığı bir metodu kendi ihtiyaçlarına göre yeniden tanımlamasıdır.

Kolay Örnek (Sınıf Tanımı)

class Hayvan {
    void sesCikar() {
        System.out.println("Hayvan sesi");
    }
}

class Kedi extends Hayvan {
    @Override
    void sesCikar() {
        System.out.println("Miyav!");
    }
}

Kullanım Örneği (Main Metodu)

public class MetotEzmeOrnek {
    public static void main(String[] args) {
        Hayvan hayvan = new Hayvan();
        hayvan.sesCikar(); // Çıktı: Hayvan sesi

        Kedi kedi = new Kedi();
        kedi.sesCikar();   // Çıktı: Miyav!
    }
}

Çıktı:

Hayvan sesi
Miyav!

Orta Örnek (Sınıf Tanımı)

class Arac {
    void calistir() {
        System.out.println("Araç çalıştı.");
    }
}

class Araba extends Arac {
    @Override
    void calistir() {
        System.out.println("Araba anahtarla çalıştı.");
    }
}

class ElektrikliAraba extends Araba {
    @Override
    void calistir() {
        System.out.println("Elektrikli araba düğmeye basılarak çalıştı.");
    }
}

Kullanım Örneği (Main Metodu)

public class OrtaMetotEzmeOrnek {
    public static void main(String[] args) {
        Arac arac1 = new Arac();
        Arac arac2 = new Araba();
        Arac arac3 = new ElektrikliAraba();

        arac1.calistir(); // Çıktı: Araç çalıştı.
        arac2.calistir(); // Çıktı: Araba anahtarla çalıştı.
        arac3.calistir(); // Çıktı: Elektrikli araba düğmeye basılarak çalıştı.
    }
}

Çıktı:

Araç çalıştı.
Araba anahtarla çalıştı.
Elektrikli araba düğmeye basılarak çalıştı.

Method Overloading (Metot Aşırı Yükleme)

Method overloading, aynı isimde farklı parametrelerle birden fazla metot tanımlamaktır.

Kolay Örnek (Sınıf Tanımı)

class Hesaplayici {
    int topla(int a, int b) {
        return a + b;
    }
    
    double topla(double a, double b) {
        return a + b;
    }
}

Kullanım Örneği (Main Metodu)

public class MetotAsiriYuklemeOrnek {
    public static void main(String[] args) {
        Hesaplayici hesaplayici = new Hesaplayici();

        System.out.println("İki int sayının toplamı: " + hesaplayici.topla(5, 10));
        System.out.println("İki double sayının toplamı: " + hesaplayici.topla(5.5, 10.5));
    }
}

Çıktı:

İki int sayının toplamı: 15
İki double sayının toplamı: 16.0

Orta Örnek (Sınıf Tanımı)

class MatematikIslemleri {

        System.out.println("İki double sayının çarpımı: " + islemler.carp(2.5, 3.0));
    }
}

Üç tamsayının çarpımı: 24

Zor Örnek (Sınıf Tanımı)

class CalisanMaasHesaplayici {
    // Temel maaş hesaplama
    double hesaplaMaas(double saatlikUcret, int calismaSaati) {
        return saatlikUcret * calismaSaati;
    }

    // Fazla mesai ile maaş hesaplama
    double hesaplaMaas(double saatlikUcret, int calismaSaati, double fazlaMesaiSaatlikUcret, int fazlaMesaiSaati) {
        return (saatlikUcret * calismaSaati) + (fazlaMesaiSaatlikUcret * fazlaMesaiSaati);
    }

    // Prim ile maaş hesaplama
    double hesaplaMaas(double temelMaas, double primOrani) {
        return temelMaas * (1 + primOrani);
    }

    // Bonus ile maaş hesaplama
    double hesaplaMaas(double saatlikUcret, int calismaSaati, double bonus) {
        return (saatlikUcret * calismaSaati) + bonus;
    }
}

Kullanım Örneği (Main Metodu)

public class ZorMetotAsiriYuklemeOrnek {
    public static void main(String[] args) {
        CalisanMaasHesaplayici maasHesaplayici = new CalisanMaasHesaplayici();

        // Temel maaş
        System.out.println("Temel Maaş: " + maasHesaplayici.hesaplaMaas(50.0, 160));

        // Fazla mesai ile maaş
        System.out.println("Fazla Mesai ile Maaş: " + maasHesaplayici.hesaplaMaas(50.0, 160, 75.0, 20));

        // Prim ile maaş
        System.out.println("Prim ile Maaş: " + maasHesaplayici.hesaplaMaas(8000.0, 0.15));

        // Bonus ile maaş

    }

Polymorphism, aynı arayüzün farklı şekillerde uygulanabilmesidir.

    double alanHesapla();
}

class Dikdortgen implements Sekil {
    private double en, boy;
    
    public Dikdortgen(double en, double boy) {
        this.en = en;
        this.boy = boy;

    @Override
    public double alanHesapla() {
        return en * boy;

}

interface Islem {

class HesaplamaIslemi implements Islem {
    @Override
    public void execute() {
        System.out.println("Hesaplama işlemi yapılıyor.");
    }

    @Override
    public void execute() {
        System.out.println("Veri kaydetme işlemi yapılıyor.");
    }
}

class RaporlamaIslemi implements Islem {
    @Override
    public void execute() {
        System.out.println("Raporlama işlemi yapılıyor.");
    }
}

// Komut deseni (Command Pattern) uygulaması
class IslemYurutucu {
    private Islem komut;

    public void setKomut(Islem komut) {
        this.komut = komut;
    }

    public void yurut() {
        if (komut != null) {
            komut.execute();
        } else {
            System.out.println("Hiçbir komut ayarlanmadı.");
        }
    }
}

Kullanım Örneği (Main Metodu)

public class ZorCokBicimlilikOrnek {
    public static void main(String[] args) {
        IslemYurutucu yurutucu = new IslemYurutucu();

        yurutucu.yurut();

        yurutucu.setKomut(new VeriKaydetmeIslemi());
        yurutucu.yurut();

        yurutucu.setKomut(new RaporlamaIslemi());
        yurutucu.yurut();

        yurutucu.setKomut(null);
        yurutucu.yurut();
    }
}

Inheritance (Kalıtım)

Inheritance, bir sınıfın başka bir sınıftan özelliklerini ve davranışlarını miras almasıdır.

Kolay Örnek (Sınıf Tanımı)

class Calisan {
    protected String isim;
    protected double maas;
    
    public Calisan(String isim, double maas) {
        this.isim = isim;
        this.maas = maas;
    }

    public void bilgileriGoster() {
        System.out.println("İsim: " + isim + ", Maaş: " + maas);
    }
}

class Yazilimci extends Calisan {
    private String programlamaDili;
    
    public Yazilimci(String isim, double maas, String programlamaDili) {
        super(isim, maas);
        this.programlamaDili = programlamaDili;
    }

    @Override
    public void bilgileriGoster() {
        super.bilgileriGoster();
        System.out.println("Programlama Dili: " + programlamaDili);
    }
}

Kullanım Örneği (Main Metodu)

public class KalitimOrnek {
    public static void main(String[] args) {
        Calisan calisan = new Calisan("Ali", 5000);
        calisan.bilgileriGoster();

        Yazilimci yazilimci = new Yazilimci("Ayşe", 7000, "Java");
        yazilimci.bilgileriGoster();
    }
}

Çıktı:

İsim: Ali, Maaş: 5000.0
İsim: Ayşe, Maaş: 7000.0
Programlama Dili: Java

abstract class Cihaz {
    protected String marka;
    protected String model;

    public Cihaz(String marka, String model) {
        this.marka = marka;
        this.model = model;
    }

    public abstract void ac();
    public abstract void kapat();

    public void bilgiGoster() {
        System.out.println("Marka: " + marka + ", Model: " + model);
    }
}

class AkilliTelefon extends Cihaz {
    private String isletimSistemi;

    public AkilliTelefon(String marka, String model, String isletimSistemi) {
        super(marka, model);
        this.isletimSistemi = isletimSistemi;
    }

    @Override
    public void ac() {
        System.out.println(marka + " " + model + " (Android) başlatılıyor...");
    }

    @Override
    public void kapat() {
        System.out.println(marka + " " + model + " (Android) kapatılıyor.");
    }

    public void uygulamaCalistir(String uygulamaAdi) {
        System.out.println(uygulamaAdi + " uygulaması " + isletimSistemi + " üzerinde çalıştırılıyor.");
    }
}

class DizustuBilgisayar extends Cihaz {
    private String islemci;

    public DizustuBilgisayar(String marka, String model, String islemci) {
        super(marka, model);
        this.islemci = islemci;
    }

    @Override
    public void ac() {
        System.out.println(marka + " " + model + " (Windows) başlatılıyor...");
    }

    @Override
    public void kapat() {
        System.out.println(marka + " " + model + " (Windows) kapatılıyor.");
    }

    public void programYukle(String programAdi) {
        System.out.println(programAdi + " programı " + islemci + " işlemcili bilgisayara yükleniyor.");
    }
}

Kullanım Örneği (Main Metodu)

public class ZorKalitimOrnek {

Encapsulation (Kapsülleme)

Encapsulation, bir nesnenin verilerini (alanlarını) ve bu verilere erişimi veya onları değiştirmeyi sağlayan metotları tek bir birim altında birleştirme ve dışarıdan doğrudan erişimi engelleme prensibidir.

Kolay Örnek (Sınıf Tanımı)

public class Ogrenci {
    private String ad;
    private int yas;

    public String getAd() {
        return ad;
    }

    public void setAd(String ad) {
        this.ad = ad;
    }

    public int getYas() {
        return yas;
    }

    public void setYas(int yas) {
        if (yas > 0) {
            this.yas = yas;
        }
    }
}

Kullanım Örneği (Main Metodu)

public class KapsullemeOrnek {
    public static void main(String[] args) {
        Ogrenci ogrenci = new Ogrenci();
        ogrenci.setAd("Fatma");
        ogrenci.setYas(20);

        System.out.println("Öğrenci Adı: " + ogrenci.getAd());
        System.out.println("Öğrenci Yaşı: " + ogrenci.getYas());

        // Geçersiz yaş denemesi
        ogrenci.setYas(-5);
        System.out.println("Yeni Öğrenci Yaşı (geçersiz): " + ogrenci.getYas());
    }
}

Çıktı:

Öğrenci Adı: Fatma
Öğrenci Yaşı: 20
Yeni Öğrenci Yaşı (geçersiz): 20

Orta Örnek (Sınıf Tanımı)

class BankaHesabi {
    private String hesapNumarasi;
    private double bakiye;

    public BankaHesabi(String hesapNumarasi, double baslangicBakiyesi) {
        this.hesapNumarasi = hesapNumarasi;
        this.bakiye = baslangicBakiyesi;
    }

    public String getHesapNumarasi() {
        return hesapNumarasi;
    }

    public double getBakiye() {
        return bakiye;
    }

    public void paraYatir(double miktar) {
        if (miktar > 0) {
            bakiye += miktar;
            System.out.println(miktar + " TL yatırıldı. Yeni bakiye: " + bakiye + " TL");
        } else {
            System.out.println("Geçersiz miktar. Para yatırma başarısız.");
        }
    }

    public void paraCek(double miktar) {
        if (miktar > 0 && bakiye >= miktar) {
            bakiye -= miktar;
            System.out.println(miktar + " TL çekildi. Yeni bakiye: " + bakiye + " TL");
        } else if (miktar <= 0) {
            System.out.println("Geçersiz miktar. Para çekme başarısız.");
        } else {
            System.out.println("Yetersiz bakiye. Para çekme başarısız.");
        }
    }
}

Kullanım Örneği (Main Metodu)

public class OrtaKapsullemeOrnek {
    public static void main(String[] args) {
        BankaHesabi hesap = new BankaHesabi("TR123456789", 1000.0);
        System.out.println("Hesap Numarası: " + hesap.getHesapNumarasi());
        System.out.println("Mevcut Bakiye: " + hesap.getBakiye() + " TL");

        hesap.paraYatir(500.0);
        hesap.paraCek(200.0);
        hesap.paraCek(1500.0); // Yetersiz bakiye
        hesap.paraYatir(-100.0); // Geçersiz miktar
    }
}

500.0 TL yatırıldı. Yeni bakiye: 1500.0 TL
200.0 TL çekildi. Yeni bakiye: 1300.0 TL
Yetersiz bakiye. Para çekme başarısız.
Geçersiz miktar. Para yatırma başarısız.

class KullaniciHesabi {
    private String kullaniciAdi;
    private String sifreHash;
    private double bakiye;

    public KullaniciHesabi(String kullaniciAdi, String sifre, double baslangicBakiyesi) {
        this.kullaniciAdi = kullaniciAdi;
        this.sifreHash = generateHash(sifre); // Şifrenin hash'i tutulur
        this.bakiye = baslangicBakiyesi;
    }

    // Getter metotları
    public String getKullaniciAdi() {
        return kullaniciAdi;
    }

    public double getBakiye() {
        return bakiye;
    }

    // Setter veya iş mantığı içeren metotlar
    public boolean sifreDegistir(String eskiSifre, String yeniSifre) {
        if (this.sifreHash.equals(generateHash(eskiSifre))) {
            this.sifreHash = generateHash(yeniSifre);
            System.out.println("Şifre başarıyla değiştirildi.");
            return true;
        } else {
            System.out.println("Eski şifre yanlış. Şifre değiştirme başarısız.");
            return false;
        }
    }

    public void paraYatir(double miktar) {
        if (miktar > 0) {
            bakiye += miktar;
            System.out.println(miktar + " TL yatırıldı. Yeni bakiye: " + bakiye + " TL");
        } else {
            System.out.println("Geçersiz miktar. Para yatırma başarısız.");
        }
    }

    public boolean paraCek(double miktar) {
        if (miktar > 0 && bakiye >= miktar) {
            bakiye -= miktar;
            System.out.println(miktar + " TL çekildi. Yeni bakiye: " + bakiye + " TL");
            return true;
        } else if (miktar <= 0) {
            System.out.println("Geçersiz miktar. Para çekme başarısız.");
            return false;
        } else {
            System.out.println("Yetersiz bakiye. Para çekme başarısız.");
            return false;
        }
    }

    // Güvenlik için basit bir hash fonksiyonu (gerçek uygulamada daha karmaşık olmalı)
    private String generateHash(String input) {
        return String.valueOf(input.hashCode());
    }
}

Abstraction (Soyutlama)

Abstraction, karmaşık sistemlerin detaylarını gizleyerek sadece önemli özelliklerini gösterme prensibidir. Java\'da soyut sınıflar ve arayüzler (interfaces) aracılığıyla uygulanır.

Kolay Örnek (Sınıf Tanımı)

abstract class Sekil {
    // Soyut metot
    public abstract double alanHesapla();

    // Normal metot
    public void bilgiGoster() {
        System.out.println("Bu bir şekildir.");
    }
}

class Daire extends Sekil {
    private double yaricap;

    public Daire(double yaricap) {
        this.yaricap = yaricap;
    }

    @Override
    public double alanHesapla() {
        return Math.PI * yaricap * yaricap;
    }
}

        daire.bilgiGoster();

        // Polymorphism ile kullanma

        System.out.println("Soyut Daire Alanı: " + soyutDaire.alanHesapla());
        soyutDaire.bilgiGoster();

Final Çalışması: Kütüphane Yönetim Sistemi

Bu örnek, öğrendiğimiz tüm OOP kavramlarını bir arada kullanarak basit bir kütüphane yönetim sistemi oluşturur.

Sınıf Tanımları

// Soyutlama (Abstraction): Temel Kitap sınıfı, soyut bir metot (getKitapTuru) içerir ve alt sınıfların bu metodu implement etmesini zorlar.
// Kapsülleme (Encapsulation): 'protected' erişim belirleyiciler ile alanları korur ve 'public' getter/setter metotları ile kontrollü erişim sağlar.
abstract class Kitap {
    protected String isim;
    protected String yazar;
    protected int sayfaSayisi;
    protected boolean oduncAlinabilir;

    public Kitap(String isim, String yazar, int sayfaSayisi) {
        this.isim = isim;
        this.yazar = yazar;
        this.sayfaSayisi = sayfaSayisi;
        this.oduncAlinabilir = true;
    }

    // Soyut metot: Tüm türetilmiş sınıflar bu metodu kendi özel tanımlamalarıyla sağlamalıdır.
    public abstract String getKitapTuru();

    // Kapsülleme: Alanlara kontrollü erişim sağlayan getter ve setter metotları
    public String getIsim() { return isim; }
    public String getYazar() { return yazar; }
    public int getSayfaSayisi() { return sayfaSayisi; }
    public boolean isOduncAlinabilir() { return oduncAlinabilir; }
    public void setOduncAlinabilir(boolean oduncAlinabilir) {
        this.oduncAlinabilir = oduncAlinabilir;
    }
}

// Kalıtım (Inheritance): Roman sınıfı, Kitap sınıfından kalıtım yoluyla türetilir.
// Metot Ezme (Method Overriding): getKitapTuru metodu, Roman sınıfına özgü bir şekilde yeniden tanımlanır.
class Roman extends Kitap {
    private String tur;

    public Roman(String isim, String yazar, int sayfaSayisi, String tur) {
        super(isim, yazar, sayfaSayisi);
        this.tur = tur;
    }

    // Metot Ezme (Override): Üst sınıftaki soyut metodu kendi özel implementasyonuyla ezer.
    @Override
    public String getKitapTuru() {
        return "Roman - " + tur;
    }
}

// Kalıtım (Inheritance): DersKitabi sınıfı da Kitap sınıfından kalıtım yoluyla türetilir.
// Metot Ezme (Method Overriding): getKitapTuru metodu, DersKitabi sınıfına özgü bir şekilde yeniden tanımlanır.
class DersKitabi extends Kitap {
    private String ders;

    public DersKitabi(String isim, String yazar, int sayfaSayisi, String ders) {
        super(isim, yazar, sayfaSayisi);
        this.ders = ders;
    }

    // Metot Ezme (Override): Üst sınıftaki soyut metodu kendi özel implementasyonuyla ezer.
    @Override
    public String getKitapTuru() {
        return "Ders Kitabı - " + ders;
    }
}

// Kapsülleme (Encapsulation): Kütüphane sınıfı, kitaplar listesini kendi içinde yönetir.
class Kutuphane {
    private List kitaplar;
    private String isim;

    public Kutuphane(String isim) {
        this.isim = isim;
        this.kitaplar = new ArrayList<>();
    }

    // Metot Aşırı Yükleme (Method Overloading): Aynı isimde iki farklı kitapEkle metodu.
    // Biri tek bir kitap eklerken, diğeri bir liste dolusu kitap ekler.
    public void kitapEkle(Kitap kitap) {
        kitaplar.add(kitap);
        System.out.println(kitap.getIsim() + " kütüphaneye eklendi.");
    }

    public void kitapEkle(List yeniKitaplar) {
        kitaplar.addAll(yeniKitaplar);
        System.out.println(yeniKitaplar.size() + " kitap kütüphaneye eklendi.");
    }

    // Çok Biçimlilik (Polymorphism): 'Kitap' tipinde bir liste üzerinden farklı türdeki kitaplara (Roman, DersKitabi) erişim.
    // Her kitap nesnesinin getKitapTuru() metodu, kendi gerçek tipine göre çağrılır.
    public void kitaplariListele() {
        System.out.println("\n" + isim + " Kütüphanesi - Kitap Listesi:");
        for (Kitap kitap : kitaplar) { // Kitap referansı, Roman veya DersKitabi nesnelerini tutabilir.
            System.out.println("Kitap: " + kitap.getIsim() + 
                             ", Yazar: " + kitap.getYazar() + 
                             ", Tür: " + kitap.getKitapTuru()); // Polymorphic metot çağrısı
        }
    }

    // İstisna İşleme (Exception Handling): try-catch blokları ile olası hataları (kitap bulunamadı, zaten ödünç alınmış) yönetir.
    public void kitapOduncAl(String kitapIsmi) {
        try {
            Kitap kitap = kitaplar.stream()
                                .filter(k -> k.getIsim().equals(kitapIsmi))
                                .findFirst()
                                .orElseThrow(() -> new Exception("Kitap bulunamadı: " + kitapIsmi));

            if (!kitap.isOduncAlinabilir()) {
                throw new Exception("Kitap zaten ödünç alınmış: " + kitapIsmi);
            }

            kitap.setOduncAlinabilir(false);
            System.out.println(kitapIsmi + " başarıyla ödünç alındı.");
        } catch (Exception e) {
            System.out.println("Hata: " + e.getMessage());
        }
    }
}

Kullanım Örneği (Main Metodu)

public class KutuphaneYonetimSistemi {
    public static void main(String[] args) {
        // Kütüphane oluştur
        Kutuphane kutuphane = new Kutuphane("Merkez Kütüphane");

        // Romanlar oluştur
        Roman roman1 = new Roman("Suç ve Ceza", "Dostoyevski", 671, "Psikolojik");
        Roman roman2 = new Roman("1984", "George Orwell", 328, "Distopik");

        // Ders kitapları oluştur
        DersKitabi dersKitabi1 = new DersKitabi("Java Programlama", "Ahmet Yılmaz", 450, "Programlama");
        DersKitabi dersKitabi2 = new DersKitabi("Veri Yapıları", "Mehmet Demir", 380, "Bilgisayar Mühendisliği");

        // Kitapları kütüphaneye ekle
        kutuphane.kitapEkle(roman1);
        kutuphane.kitapEkle(roman2);
        kutuphane.kitapEkle(dersKitabi1);
        kutuphane.kitapEkle(dersKitabi2);

        // Kitapları listele
        kutuphane.kitaplariListele();

        // Kitap ödünç alma işlemleri
        kutuphane.kitapOduncAl("Suç ve Ceza");
        kutuphane.kitapOduncAl("Suç ve Ceza"); // Aynı kitabı tekrar ödünç alma denemesi
        kutuphane.kitapOduncAl("Bilinmeyen Kitap"); // Olmayan kitabı ödünç alma denemesi
    }
}
